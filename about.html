Описание принципов работы программного продукта и стека технологий
Программный продукт необходим для расчета оптимального кол-во производства и поставки продукции при работе с маркетплейсами Widdberries и Ozon.
Этапы взаимодействия пользователя с системой:
  • На этапе создания аккаунта пользователь передаёт в систему посредством предоставления доступа к API ключу данные об остатках товара, динамике продаж и других статистических и коммерческих показателях кабинета продавца на маркетплейсе.
  • Помимо данных из API пользователя для оптимизации цепочки поставок используются данные собираемые в системе с помощью парсинга общедоступной информации с помощью разработанных нами роботов.
  • Дополнительно в системе настроена интеграция с учетными системами товарного учёта с помощью загрузки\выгрузки файлов в формате xls. Пользователь может передать в систему данные об остатках на своих складах, кратности отгрузки продукции и другие параметры.
  • Анализируя полученную информацию система может прогнозировать темпы расхода продукции, учитывать сезонность продаж и изменение темпов реализации для сохранения баланса между наличием товара на складах и кол-ве вложенных в товары средств.
  • Задавая целевые показатели по планируемой оборачиваемости, кол-ве дней на которое нужно произвести продукцию, дате отгрузки и прочие, пользователь по результатам расчёта получает рекомендуемые параметры к производству или отгрузке продукции на маркетплейс. Данные предоставляются в табличном виде, удобные для анализа лица принимающего решение.
  • Проверив данные и внеся необходимые изменения, по результатам взаимодействия с системой пользователь получает данные в удобном для дальнейшей обработки виде, это выгрузка в формате xls подходящая для загрузки в ЛК маркетплейса или в свою товаро-учетную систему.
Используемые технологии:
  • PHP
  • MySQL
  • Laravel
  • Redis, Memcache
  • JS, HTML5, CSS
